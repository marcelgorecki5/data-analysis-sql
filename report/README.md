# Analiza danych sprzedażowych – SQL

## Opis
Projekt analityczny oparty na relacyjnej bazie danych sprzedażowych.
Celem było przygotowanie raportów biznesowych na podstawie zapytań SQL.

## Zakres projektu
- projekt bazy danych (klienci, zamówienia, produkty)
- zapytania SQL (JOIN, GROUP BY, CTE)
- analiza danych sprzedażowych i KPI

## Technologie
- SQL (MySQL / PostgreSQL)
- Excel / Power BI

## Przykładowe analizy
- sprzedaż miesięczna
- top produkty
- średnia wartość zamówienia
